# Scoop

## Install
### Script
```
iwr -useb get.scoop.sh | iex
```

## CLI apps
### Utility
```
scoop install curl sudo jq

// 'jq' suggests installing 'jid'.
scoop install jid
```

### Neovim
```
scoop install neovim gcc

// 'neovim' suggests installing 'extras/vcredist2022'.
scoop install extras/vcredist2022
```
